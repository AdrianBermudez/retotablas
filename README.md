# Ejercicio de tablas con HTML
El proyecto “m1ra1ahd02” consiste en crear una carpeta que contenga todos los archivos necesarios para el funcionamiento de un sitio web desarrollado en HTML. Dentro de este proyecto se deben construir las páginas solicitadas y, una vez finalizado, publicar el sitio en un servicio de hosting gratuito para compartir su enlace.
